VR Education Project

This VR project is targeting to teenagers who want to
learn about chemical compounds in an interactive environment while having fun

Having Fun

Users can teleport and walk around the map. They can also climb to the second floor if they want to get access to more molecule models

Learning Panel

There are floating interactive windows that will pop out when detecting nearby users and demonstrate specific molecule information

Progression System

There are multiple challenges that allow users to test themselves in integrated progressive levels and consolidate their knowledge
